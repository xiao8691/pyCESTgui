🎉 项目概述
==============
这是一个功能完整、可直接使用的CEST (Chemical Exchange Saturation Transfer) MRI
图像处理工具。提供直观的图形界面和强大的数据分析功能。

✅ 开发状态: 已完成
📅 完成日期: 2026年3月11日
💾 项目位置: e:\01_CEST\pyCESTgui

========================================================================================
🚀 快速开始 (3步启动)
========================================================================================

方法1: 运行Python脚本 (推荐 - 需要Python 3.8+)
    1. 打开命令提示符或PowerShell
    2. 进入项目目录: cd e:\01_CEST\pyCESTgui
    3. 运行脚本: python run.py
    4. 选择选项 "1" 启动GUI

方法2: 直接运行Python应用
    python main.py

方法3: 使用可执行文件 (无需Python环境)
    1. 生成.exe: python build_exe.py
    2. 运行dist/CESTGui.exe

💡 核心功能说明
========================================================================================

✓ 数据导入
  - NIfTI格式CEST MRI数据加载
  - ROI mask加载和验证
  - 化学位移偏移量(ppm)导入

✓ 预处理选项
  - PCA降噪 (可配置保留比例)
  - 配准校正 (平移和旋转)
  - B0不均匀性校正
  - 高斯平滑和中值滤波

✓ CEST拟合分析
  - 2-Step Lorentzian拟合
  - 自动B0校正
  - 支持8种代谢产物分析:
    ✓ Water        ✓ MT          ✓ Amide
    ✓ NOE(-3.5)    ✓ Creatine    ✓ NOE(-1.6)
    ✓ Amine        ✓ Hydroxyl

✓ 结果显示和导出
  - Z-Spectroscopy曲线可视化
  - 代谢产物百分比参数图
  - ROI区域z谱拟合曲线
  - CSV格式数据导出

📋 使用步骤
========================================================================================

步骤1: 准备数据
  - 获取NIfTI格式CEST数据 (*.nii.gz)
  - 准备ROI mask文件
  - 准备频率偏移文本文件 (ppm值)
  或使用: example_data中的示例文件

步骤2: 启动应用
  python main.py
  或 python run.py → 选择 "1"

步骤3: 加载数据
  - 点击"CEST数据"旁的"浏览"按钮加载数据
  - 点击"Mask"旁的"浏览"按钮加载mask
  - 点击"Offset"旁的"浏览"按钮加载offsets

步骤4: 配置分析参数
  - 勾选要分析的代谢产物
  - 可选: 启用PCA降噪、高斯平滑
  - 默认启用B0校正

步骤5: 执行分析
  - 点击"执行拟合"按钮
  - 等待处理完成(进度条显示) 
  - 查看Z谱标签页的拟合结果

步骤6: 查看和导出结果
  - 在"Z谱"标签页查看拟合曲线
  - 在"结果表格"查看定量参数
  - 点击"导出结果"保存CSV文件

🐛 常见问题快速解决
========================================================================================

Q: 如何生成.exe可执行文件?
A: 运行 python build_exe.py
   输出文件位置: dist/CESTGui.exe

Q: 什么时候使用" ROI平均"模式?
A: 分析单个或多个感兴趣区域(ROI)的平均z谱
   快速、稳定、推荐初学者使用

Q: 什么时候使用"逐像素分析"?
A: 生成参数图，显示整个图像中的代谢产物分布
   耗时较长，需要更好的数据质量

Q: 拟合失败怎么办?
A: 1. 尝试启用PCA降噪(75-85%)
   2. 检查数据格式是否为NIfTI
   3. 确认offset和数据维度匹配
   4. 查看日志窗口获取详细错误信息

Q: 如何联系技术支持?
A: 查看日志窗口的错误信息
   或参考README.md中的故障排除部分
